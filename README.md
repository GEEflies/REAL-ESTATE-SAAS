# Aurix

**Professional AI Real Estate Photo Enhancement**

Aurix is a cutting-edge SaaS platform designed to help real estate agents and photographers transform their property photos instantly. Using advanced AI models, Aurix automates complex editing tasks such as HDR merging, sky replacement, window pulling, and object removal, delivering professional results in seconds.

## üöÄ Features

- **AI Enhancement**: Automatically correct exposure, color, and lighting to professional standards.
- **Sky Replacement**: Turn gloomy days into perfect weather with realistic, AI-generated skies.
- **Window Pulling**: Fix overexposed windows to clearly reveal the view outside.
- **Privacy Protection**: Automatic detection and blurring of license plates and faces.
- **Batch Processing**: Enhance entire property listings in one go with our efficient queue system.
- **Multi-language Support**: Fully localized interface available in English and Slovak.

## üì¶ Getting Started

1.  **Clone the repository**
    ```bash
    git clone https://github.com/GEEflies/REAL-ESTATE-SAAS.git
    ```

2.  **Install dependencies**
    ```bash
    cd nana-banana-pro
    npm install
    ```

3.  **Set up environment variables**
    Create a `.env.local` file in the root directory and configure your API keys for:
    - Supabase (URL & Anon Key)
    - Google Gemini
    - Stripe
    - Vercel Blob

4.  **Run the development server**
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) to view the application.
